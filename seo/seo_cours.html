L'optimisation pour les moteur de recherche, ou SEO (pourSearch engine optimisation ), est un ensemble de techniques pour optimiser la visibilité une page web les page du résultats de recherche (les SERP, pour search Engine result Pages)

L'objectif est d'améliorer le positionnement d'une page web dans les page de résultats de recherche sur des mots-clés cibles (selon les thémes principaux du site). On considére que le positionnement d'un site est bon lorsqu'il est positionné (classé) dans la prémier réponses naturelles d'une recheche sur des mots-clés correspondant à sa thémùatique.
